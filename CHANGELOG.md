# Changelog

## 0.8.3.0

- Added “Execute on Session End” automation option to Execute Command action
- Improved rendering of Execute Command action on both key and dial controllers

## 0.8.0.0

- Added new action: Execute UI Command (Beta)
- Added automation options to the Execute Command action
- Added visual orientation option to the Trigger Blueprint action
- Fixed issue where settings were not correctly initialized for the Adjust Editor Volume action

## 0.7.0.0

- Added support for Unreal Engine 5.7
- Added new action: Adjust Editor Volume
- Added new action: Play World
- Added visual feedback for success/failure to the Execute Command and Trigger Blueprint actions
- Added the current value of the selected console variable to the Execute Command action display
- Added visual customization options to the Execute Command and Trigger Blueprint actions
- Improved display layout behavior of the Execute Command and Trigger Blueprint actions
- Improved readability of property inspector status indicators
- Improved behavior of suggestions provided by the Execute Command property inspector

## 0.6.0.0

- Added a throttle to prevent Set Value from flooding the Stream Deck connection with messages

## 0.5.0.0

- Initial release
